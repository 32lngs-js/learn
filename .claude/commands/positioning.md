# Brand Positioning Workshop

Develop a complete brand positioning using structured frameworks.

## Inputs

- **Brand/product name** (if decided)
- **Target audiences** (who are the primary and secondary users?)
- **Key competitors** (who are you compared against?)
- **User research** (any persona work, interviews, or data from /persona)

If the user hasn't done persona or competitor work yet, suggest running those first but proceed with available information.

## Process

### Phase 1 — Brand Pyramid (Bottom-Up)

Build the pyramid layer by layer:

1. **Features** — List the product's key features
   - Interactive lessons, spaced repetition quizzes, AI practice interviews, Sparks currency, achievements, streaks, drill-based mastery, multi-course catalog

2. **Functional Benefits** — What do these features do for the user?
   - Faster skill acquisition, realistic practice, measurable progress, habit formation

3. **Emotional Benefits** — How does the user feel?
   - Confident, prepared, accomplished, in control of their growth

4. **Brand Personality** — If the brand were a person, how would they act?
   - (Define 3-5 personality traits)

5. **Brand Essence** — The single core idea (2-3 words max)
   - (Distill everything into the irreducible truth)

### Phase 2 — Positioning Statement

Fill in the template:

> **For** [target audience]
> **who** [situation / need / JTBD]
> **[Product name]** is a [category]
> **that** [key benefit / promise]
> **Unlike** [primary alternative/competitor]
> **we** [unique differentiator]

Generate 3 variants of this statement and evaluate each.

### Phase 3 — Messaging Pillars

Define 3-4 messaging pillars that support the positioning:
- Each pillar = a key theme the brand consistently communicates
- For each pillar: one-line summary, proof points, example copy

Example pillars:
- "Learn by doing" (active practice, not passive watching)
- "AI that coaches you" (personalized feedback, interview practice)
- "Progress you can feel" (sparks, streaks, mastery levels)

### Phase 4 — Voice & Tone Guide

Define how the brand communicates:
- **Voice** (consistent personality): e.g., knowledgeable but never condescending, encouraging but honest
- **Tone variations** by context:
  - Onboarding: warm, excited
  - Error states: empathetic, helpful
  - Achievement: celebratory, proud
  - Marketing: confident, inspiring
- **Words we use** vs **Words we avoid**
- **Example rewrites** (take a generic sentence and rewrite it in brand voice)

### Phase 5 — Competitive Positioning Map

Create a 2x2 positioning map with:
- X-axis: Passive learning ←→ Active practice
- Y-axis: Broad catalog ←→ Deep mastery
- Plot competitors and our product

Identify the "white space" where no competitor sits.

Present the complete positioning as a one-page strategy brief.
