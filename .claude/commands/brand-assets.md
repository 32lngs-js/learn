# Brand Asset Generator

Create creative briefs and specifications for all brand assets needed to launch.

## Inputs

- **Visual identity** (run /visual-identity first if not done — need colors, fonts, style direction)
- **Brand name and tagline**
- **Priority assets** (what do you need first? logo, social, website, app store, all)

If the user doesn't provide these, ask what's most urgent.

## Process

### Phase 1 — Asset Inventory

List every asset needed, categorized:

**Identity Core**
- [ ] Primary logo (SVG, PNG @1x, @2x, @3x)
- [ ] Logo variations (horizontal, stacked, icon-only, monochrome, reversed)
- [ ] Favicon (16x16, 32x32, 180x180 apple-touch, 192x192 android)
- [ ] PWA icons (192x192, 512x512, maskable)
- [ ] Brand color swatches file

**Website / App**
- [ ] OG image template (1200x630) — for link previews on social
- [ ] Hero section imagery / illustration
- [ ] Feature illustrations (3-5 key features)
- [ ] Empty state illustrations
- [ ] Loading/skeleton states
- [ ] Error page illustration (404, 500)
- [ ] Onboarding screens (3-5 slides)

**Social Media**
- [ ] Profile picture (square, works at 40x40)
- [ ] Twitter/X banner (1500x500)
- [ ] LinkedIn banner (1584x396)
- [ ] Instagram profile & highlight covers
- [ ] Post templates (quote card, stat card, tip card, course promo)
- [ ] Story/reel templates

**Email**
- [ ] Email header/logo lockup
- [ ] Welcome email template
- [ ] Weekly digest template
- [ ] Achievement notification template
- [ ] Re-engagement email template

**Marketing**
- [ ] Pitch deck template (Google Slides / Keynote)
- [ ] One-pager / sell sheet (PDF)
- [ ] Case study template
- [ ] Blog post header template

**App Store (if applicable)**
- [ ] App icon (1024x1024)
- [ ] Screenshots (6.7" and 5.5" iPhone, iPad, Android)
- [ ] Feature graphic (1024x500 for Google Play)
- [ ] Short description (80 chars) and full description

### Phase 2 — Creative Briefs

For each priority asset, write a creative brief:
- **Asset name**
- **Dimensions / format**
- **Purpose** (where it appears, who sees it)
- **Content** (text, imagery, elements to include)
- **Brand guidelines to follow** (colors, fonts, logo placement)
- **Examples / references** (links to similar assets from competitors or inspiration)
- **Do's and Don'ts**

### Phase 3 — DIY vs Hire

For each asset, recommend:
- **DIY tools** (Figma, Canva, DALL-E, Midjourney) — for quick/budget execution
- **Hire** (freelance designer, agency) — for professional quality
- **Estimated cost** (DIY: $0-50, Freelance: $100-500, Agency: $500-5000)
- **Priority** (Must-have for launch / Nice-to-have / Can wait)

### Phase 4 — Production Schedule

Create a prioritized production timeline:
- **Week 1:** Logo, favicon, brand colors finalized
- **Week 2:** OG image, social profiles, email header
- **Week 3:** Website hero, feature illustrations
- **Week 4:** Social templates, pitch deck, app store assets

Present the full inventory with briefs as a project plan a designer can execute.
