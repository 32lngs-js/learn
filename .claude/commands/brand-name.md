# Brand Name Generator

Generate, evaluate, and shortlist name candidates for the platform, a feature, or a course.

## Inputs

- **What are we naming?** (the whole platform, a feature like "Sparks", a course, a campaign)
- **Audience:** Who will see/say this name?
- **Vibe keywords:** 3-5 adjectives that describe the desired feeling (e.g., playful, expert, approachable, bold)
- **Constraints:** Any names to avoid, syllable limits, domain TLD preferences (.com, .io, .ai, etc.)

If the user doesn't provide these, ask before proceeding.

## Process

### Phase 1 — Generate Candidates (30+)

Use multiple naming strategies:
1. **Descriptive** — says what it does (e.g., SkillForge, LearnLoop)
2. **Metaphorical** — evokes a feeling/concept (e.g., Kindling, Meridian)
3. **Invented/portmanteau** — coined words (e.g., Cognify, Practium)
4. **Action-oriented** — verbs/imperatives (e.g., Ignite, Upcraft)
5. **Acronym/abbreviation** — (e.g., LEAP, ARC)
6. **Cultural/mythological** — references that fit (e.g., Athena, Forge)

Generate at least 30 candidates across these categories.

### Phase 2 — Evaluate

Score each candidate on:
- **Memorability** (1-5): Is it easy to recall and spell?
- **Relevance** (1-5): Does it connect to learning/growth/mastery?
- **Uniqueness** (1-5): Is it distinctive in the edtech space?
- **Domain potential** (1-5): Is a .com/.io/.ai likely available?
- **Say-aloud test** (1-5): Does it sound good spoken?
- **Scalability** (1-5): Will it still fit if the product expands?

### Phase 3 — Domain & Handle Check

For the top 10 candidates:
- Search for domain availability (.com, .io, .ai, .co)
- Check if obvious social handles would conflict (Twitter/X, LinkedIn, GitHub)
- Flag any obvious trademark conflicts with major brands

Use web search tools to verify availability.

### Phase 4 — Shortlist

Present a ranked shortlist of 5 finalists with:
- Name
- Available domains
- Tagline suggestion that pairs with it
- One-line rationale (why this name works for the brand)

Ask the user to pick their top 2-3 for deeper exploration.
