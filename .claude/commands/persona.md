# User Persona Builder

Create a detailed, actionable user persona grounded in Jobs-to-be-Done research.

## Inputs

- **Persona type** (career switcher, student, professional, hobbyist, team lead, etc.)
- **Course focus** (which courses would they use? AI fluency, CFA, poker, coding, etc.)
- **Any real user data** (interviews, analytics, support tickets, user emails you've observed)

If the user doesn't provide these, ask what segment to focus on.

## Process

### Phase 1 — JTBD Discovery

Define the core "job" this persona hires the product to do:
- **Main job:** "When I [situation], I want to [motivation], so I can [desired outcome]."
- **Related jobs:** 2-3 adjacent jobs they're also trying to accomplish
- **Current solutions:** What do they use today? (YouTube, textbooks, Udemy, Coursera, nothing)
- **Why current solutions fail:** What's frustrating about the alternatives?

### Phase 2 — Persona Profile

Build the persona card:
- **Name** (fictional but realistic)
- **Age, location, role/occupation**
- **Background** (education, career stage, tech comfort level)
- **Goals** (what does success look like for them in 6 months?)
- **Frustrations** (what blocks them from achieving their goals?)
- **Quote** (a one-liner that captures their mindset)
- **Day in the life** (when/where/how would they use the platform?)

### Phase 3 — Product Mapping

Map this persona to platform features:
- Which courses are most relevant?
- Which features matter most? (AI interviews, SRS quizzes, Sparks, streaks, drills)
- What's their likely journey? (discovery → signup → first lesson → habit → paid?)
- What would make them tell a friend?
- What would make them churn?

### Phase 4 — Messaging Angle

Write:
- **One-liner pitch** tailored to this persona
- **3 value props** that would resonate with them specifically
- **Objections** they'd raise and how to counter them
- **Social proof** they'd find compelling (testimonials, stats, credentials)

### Phase 5 — Acquisition Channel

Recommend:
- Where does this persona hang out online? (Reddit, Twitter/X, LinkedIn, Discord, specific forums)
- What search terms would they use?
- What content would attract them? (blog posts, videos, tools, challenges)
- What ad copy would stop their scroll?

Present the complete persona as a one-page reference card.
